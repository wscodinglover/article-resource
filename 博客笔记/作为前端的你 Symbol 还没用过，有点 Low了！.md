## Symbol 是什么

**Symbol 是一种基本数据类型**。`Symbol()` 函数返回 `symbol`类型的值，

> 注意！注意！注意！ 通过 `Symbol`创建返回的 `symbol` 值都是**唯一**的。一个`symbol` 值能作为对象属性的标识符；这是该**数据类型**仅有的目的。

创建 `symbol` 值的语法（注意没有 `new` ）：

```
<span data-darkreader-inline-color="">Symbol</span>([description])<br>
```

举个例子：

```
<span data-darkreader-inline-color="">const</span>&nbsp;symbol1&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>();<br><span data-darkreader-inline-color="">const</span>&nbsp;symbol2&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">123</span>);<br><span data-darkreader-inline-color="">const</span>&nbsp;symbol3&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'test'</span>);<br><br><span data-darkreader-inline-color="">console</span>.log(symbol2);&nbsp;<span data-darkreader-inline-color="">//&nbsp;Symbol(123)</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">typeof</span>&nbsp;symbol1);&nbsp;<span data-darkreader-inline-color="">//&nbsp;symbol</span><br><span data-darkreader-inline-color="">console</span>.log(symbol2&nbsp;===&nbsp;<span data-darkreader-inline-color="">123</span>);&nbsp;<span data-darkreader-inline-color="">//&nbsp;这个永远输出是false</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'test'</span>)===<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'test'</span>));<span data-darkreader-inline-color="">//&nbsp;永远输出是false</span><br>
```

可以看出通过 `Symbol` 创建返回的`symbol`值都是唯一的。

## Symbol 特点说明

这些特点可能会导致你在开发过程中出现 `BUG` ，需要知道下。

### 1\. Symbol 不可以使用New创建

```
<span data-darkreader-inline-color="">var</span>&nbsp;symbol&nbsp;=&nbsp;<span data-darkreader-inline-color="">new</span>&nbsp;<span data-darkreader-inline-color="">Symbol</span>()<br>
```

上面这种是有问题的创建 `symbol` 值方式，会抛出 `TypeError` 错误。

### 2\. Symbol类型值需通过 Object.getOwnPropertySymbols() 获取

`Symbol` 创建的值是不可枚举的，以下方式遍历对象的结果都不会包含 `symbol` 内容

-   `for in` 循环:循环会遍历对象的可枚举属性，但会忽略不可枚举的属性
    
-   `Object.keys()`:方法返回一个数组，其中包含对象的所有可枚举属性的名称。不可枚举的属性不会被包含在返回的数组中
    
-   `JSON.stringify()`:只会序列化对象的可枚举属性，而不会包含不可枚举属性
    
-   `Object.assign()`:用于将源对象中可枚举属性复制到目标对象。不可枚举属性不会复制
    
-   `Object.getOwnPropertyNames()`:返回一个数组，其中包含对象的所有属性（**包括不可枚举属性**）的名称，但是**不包括使用`symbol`值**作为名称的属性
    

可以将 `Symbol` 类型数据遍历出来的函数

1.  `Object.getOwnPropertySymbols` 方法可以获取指定对象的所有 `Symbol` 属性名
    
2.  `Reflect.ownKeys` 方法可以获取指定对象的所有`Symbol` 属性名
    

```
<span data-darkreader-inline-color="">let</span>&nbsp;symbol&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'test'</span>);<br><span data-darkreader-inline-color="">let</span>&nbsp;obj&nbsp;=&nbsp;{[symbol]:<span data-darkreader-inline-color="">123</span>};<br><span data-darkreader-inline-color="">for</span>(<span data-darkreader-inline-color="">const</span>&nbsp;key&nbsp;<span data-darkreader-inline-color="">in</span>&nbsp;obj){<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">console</span>.log(key);&nbsp;<span data-darkreader-inline-color="">//&nbsp;无打印信息</span><br>}<br><span data-darkreader-inline-color="">console</span>.log(obj[symbol]);&nbsp;<span data-darkreader-inline-color="">//&nbsp;123</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Object</span>.keys(obj));&nbsp;<span data-darkreader-inline-color="">//&nbsp;[]</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Object</span>.getOwnPropertySymbols(obj));&nbsp;<span data-darkreader-inline-color="">//&nbsp;[&nbsp;Symbol(test)&nbsp;]</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Reflect</span>.ownKeys(obj));&nbsp;<span data-darkreader-inline-color="">//&nbsp;[&nbsp;Symbol(test)&nbsp;]</span><br>
```

3.  `Object.assign` 将属性从源对象复制到目标对象，会包含 `Symbol` 类型作为 `key` 的属性
    

```
<span data-darkreader-inline-color="">const</span>&nbsp;symbolKey&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'key'</span>);<br><span ### 3\. Symbol 在 JSON.stringify 中的问题

`JSON.stringify` 转换和 `Symbol`相关的数据时，规则如下

-   `JSON.stringify` 的时候，如果对象中 `key` 或者 `value`都是 `Symbol`类型时候。转换过程会把它忽略掉
    
-   `JSON.stringify` 直接转换 `symbol`类型数据,转换后的结果为 `undefined`
    
-   `JSON.stringify` 转换的是一个对象，无论 `key` 还是 `value`中有 `symbol`类型，都会忽略掉
    

### 4\. 对象中声明的 Symbol 属性获取方式

需要使用 `Symbol` 对象属性的时候，用 `obj[]`  ,必须使用`[]`方式获取属性

```
<span data-darkreader-inline-color="">let</span>&nbsp;symbol&nbsp;=&nbsp;<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">'test'</span>);<br><span data-darkreader-inline-color="">let</span>&nbsp;obj&nbsp;=&nbsp;{[symbol]:<span data-darkreader-inline-color="">123</span>};<br><span data-darkreader-inline-color="">console</span>.log(obj[symbol]);&nbsp;<span data-darkreader-inline-color="">//&nbsp;123</span><br>
```

### 5\. Symbol.For 介绍

项目中如果想要使用同一个 `Symbol` 值，可以使用 `Symbol.for`。它接受一个字符串作为参数，用 `Symbol.for()` 方法创建的 `symbol` 会被放入一个全局的 `symbol` 注册表。`Symbol.for()`不是每次都创建新的 `symbol`，会先搜索有没有以该参数作为名称的 `Symbol` 值。如果有，就返回这个 `Symbol` 值，否则就新建并返回一个以该字符串为名称的 `Symbol` 值。

```
<span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Symbol</span>.for(<span data-darkreader-inline-color="">'test'</span>)&nbsp;===&nbsp;<span data-darkreader-inline-color="">Symbol</span>.for(<span data-darkreader-inline-color="">'test'</span>));&nbsp;<span data-darkreader-inline-color="">//&nbsp;true</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">"test"</span>)===<span data-darkreader-inline-color="">Symbol</span>(<span data-darkreader-inline-color="">"test"</span>));&nbsp;<span data-darkreader-inline-color="">//&nbsp;false</span><br>
```

### 6\. Symbol 直接传入一个函数

使用 `Symbol` 直接传入一个函数，会调用 `toString` 函数，将函数内容转换为字符串

```
<span data-darkreader-inline-color="">let</span>&nbsp;a&nbsp;=&nbsp;<span><span data-darkreader-inline-color="">function</span>(<span></span>)</span>{<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">'哈哈哈'</span>);<br>}<br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">Symbol</span>(a));&nbsp;<span data-darkreader-inline-color="">//&nbsp;&nbsp;Symbol(function(){&nbsp;&nbsp;&nbsp;console.log('哈哈哈');})</span><br><span data-darkreader-inline-color="">console</span>.log(<span data-darkreader-inline-color="">typeof</span>&nbsp;<span data-darkreader-inline-color="">Symbol</span>(a));&nbsp;<span data-darkreader-inline-color="">//&nbsp;symbol</span><br>
```

## Symbol 应用场景

### Symbol.toStringTag 使用

`Symbol.toStringTag` 官方描述是一个字符串值属性，用于创建对象的默认字符串描述。由 `Object.property.toString()` 方法内部访问

`MDN` 描述: `Object.property.toString()` 返回一个表示该对象的字符串。旨在重写(自定义)派生类对象的类型转换逻辑。

> **它在我开发过程中最常用场景是判断类型**

举个例子:

```
<span data-darkreader-inline-color="">const</span>&nbsp;toStringCallFun&nbsp;=&nbsp;<span data-darkreader-inline-color="">Object</span>.prototype.toString.call;<br>toStringCallFun(<span data-darkreader-inline-color="">new</span>&nbsp;<span data-darkreader-inline-color="">Date</span>);&nbsp;<span data-darkreader-inline-color="">//&nbsp;[object&nbsp;Date]</span><br>toStringCallFun(<span data-darkreader-inline-color="">new</span>&nbsp;<span data-darkreader-inline-color="">String</span>);&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;[object&nbsp;String]</span><br>toStringCallFun(<span data-darkreader-inline-color="">Math</span>);&nbsp;<span data-darkreader-inline-color="">//&nbsp;[object&nbsp;Math]</span><br>toStringCallFun(<span data-darkreader-inline-color="">undefined</span>);&nbsp;<span data-darkreader-inline-color="">//&nbsp;[object&nbsp;Undefined]</span><br>toStringCallFun(<span data-darkreader-inline-color="">null</span>);&nbsp;<span data-darkreader-inline-color="">//&nbsp;[object&nbsp;Null]</span><br>
```

我们也可以自定义函数，来覆盖默认的 `toString()` 方法。自定义的 `toString()` 方法可以是任何你想要的值。 举个例子:

```
<span><span data-darkreader-inline-color="">function</span>&nbsp;<span data-darkreader-inline-color="">Student</span>(<span>score,age</span>)</span>{<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">this</span>.score&nbsp;=&nbsp;score;<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">this</span>.age&nbsp;=&nbsp;age;<br>}<br><span data-darkreader-inline-color="">let</span>&nbsp;student&nbsp;=&nbsp;<span data-darkreader-inline-color="">new</span>&nbsp;Student(<span data-darkreader-inline-color="">'100'</span>,<span data-darkreader-inline-color="">13</span>);<br><span data-darkreader-inline-color="">//&nbsp;直接调用toString函数</span><br><span data-darkreader-inline-color="">console</span>.log(student.toString());&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Object]'</span><br><br><span data-darkreader-inline-color="">//&nbsp;覆盖默认的toString函数</span><br>Student.prototype.toString&nbsp;=&nbsp;<span><span data-darkreader-inline-color="">function</span>(<span></span>)</span>{<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">return</span>&nbsp;<span data-darkreader-inline-color="">`年龄<span data-darkreader-inline-color="">${<span data-darkreader-inline-color="">this</span>.age}</span>;成绩<span data-darkreader-inline-color="">${<span data-darkreader-inline-color="">this</span>.score}</span>`</span><br>}<br><span data-darkreader-inline-color="">console</span>.log(student.toString());&nbsp;<span data-darkreader-inline-color="">//&nbsp;年龄13;成绩100</span><br>
```

默认情况下，`toString()` 方法被每个 `Object` 对象继承，如果此方法在自定义对象中未被覆盖，`toString()` 返回`“[object type]”`，其中 `type` 是对象的类型。

`Symbol.toStringTag` 官方已经说了它定义了 `Object.prototype.toString()` 方法的返回值。在`ES6` 之后大多数内置的对象提供了它们自己的 `Symbol.toStringTag` 标签，`toString` 时回默认返回 `Symbol.toStringTag` 键对应的值。比如常见的如下

```
<span data-darkreader-inline-color="">Object</span>.prototype.toString.call(<span data-darkreader-inline-color="">new</span>&nbsp;<span data-darkreader-inline-color="">Map</span>());&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;"[object&nbsp;Map]"</span><br><span data-darkreader-inline-color="">Object</span>.prototype.toString.call(<span><span data-darkreader-inline-color="">function</span>*&nbsp;(<span></span>)&nbsp;</span>{});&nbsp;<span data-darkreader-inline-color="">//&nbsp;"[object&nbsp;GeneratorFunction]"</span><br><span data-darkreader-inline-color="">Object</span>.prototype.toString.call(<span data-darkreader-inline-color="">Promise</span>.resolve());&nbsp;<span data-darkreader-inline-color="">//&nbsp;"[object&nbsp;Promise]"</span><br><span data-darkreader-inline-color="">//&nbsp;...&nbsp;and&nbsp;more</span><br>
```

但是在早期不是所有对象都有 `toStringTag` 属性，没有 `toStringTag` 属性的对象也会被`toString()` 方法识别并返回特定的类型标签。如下：

```
<span data-darkreader-inline-color="">let</span>&nbsp;toStringFunc&nbsp;=&nbsp;<span data-darkreader-inline-color="">Object</span>.prototype.toString<br>toStringFunc.call(<span data-darkreader-inline-color="">'foo'</span>)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;String]'</span><br>toStringFunc.call([<span data-darkreader-inline-color="">1</span>,&nbsp;<span data-darkreader-inline-color="">2</span>])&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Array]'</span><br>toStringFunc.call(<span data-darkreader-inline-color="">3</span>)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Number]'</span><br>toStringFunc.call(<span data-darkreader-inline-color="">true</span>)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Boolean]'</span><br>toStringFunc.call(<span data-darkreader-inline-color="">undefined</span>)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Undefined]'</span><br>toStringFunc.call(<span data-darkreader-inline-color="">null</span>)&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;Null]'</span><br>

介绍了这么多再来说我们可以 `Symbol.toStringTag` 做点什么? 我们自己创建的类，`toString()` 可就找不到 `toStringTag` 属性喽！只会默认返回 `Object` 标签

```

我们给类增加一个 `toStringTag` 属性，自定义的类也就拥有了自定义的类型标签

```
<span><span data-darkreader-inline-color="">class</span>&nbsp;<span data-darkreader-inline-color="">TestClass</span></span>{<br>&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">get</span>&nbsp;[Symbol.toStringTag](){<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<span data-darkreader-inline-color="">return</span>&nbsp;<span data-darkreader-inline-color="">"TestToStringTag"</span><br>&nbsp;&nbsp;&nbsp;&nbsp;}<br>}<br><span data-darkreader-inline-color="">Object</span>.prototype.toString.call(<span data-darkreader-inline-color="">new</span>&nbsp;TestClass());<span data-darkreader-inline-color="">//&nbsp;'[object&nbsp;TestToStringTag]'</span><br>
