## 扩展知识：vue3 使用具名插槽

在 Vue 3 中，你可能会遇到“具名插槽”（Named Slots）的概念，它允许你在组件内部定义特定的插槽位置，并在父组件中为这些具名插槽提供内容。

以下是如何在Vue 3中使用具名插槽的一个基本示例：

1.  1. 首先，在子组件（ChildComponent.vue）中定义一个具名插槽：
    

```
&lt;<span data-darkreader-inline-color="">template</span>&gt;<br>&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">div</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">h2</span>&gt;这是子组件的标题&lt;/<span data-darkreader-inline-color="">h2</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">slot</span>&nbsp;<span data-darkreader-inline-color="">name</span>=<span data-darkreader-inline-color="">"description"</span>&gt;默认描述内容&lt;/<span data-darkreader-inline-color="">slot</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">slot</span>&nbsp;<span data-darkreader-inline-color="">name</span>=<span data-darkreader-inline-color="">"footer"</span>&gt;默认页脚内容&lt;/<span data-darkreader-inline-color="">slot</span>&gt;<br>&nbsp;&nbsp;&lt;/<span data-darkreader-inline-color="">div</span>&gt;<br>&lt;/<span data-darkreader-inline-color="">template</span>&gt;
```

在这个例子中，我们定义了两个具名插槽：`description` 和 `footer`，并且都提供了默认内容。

1.  1. 然后，在父组件中使用这个子组件并填充具名插槽：
    

```
&lt;<span data-darkreader-inline-color="">template</span>&gt;<br>&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">ChildComponent</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">template</span>&nbsp;<span data-darkreader-inline-color="">v-slot:description</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">p</span>&gt;这是来自父组件的描述内容&lt;/<span data-darkreader-inline-color="">p</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/<span data-darkreader-inline-color="">template</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">template</span>&nbsp;<span data-darkreader-inline-color="">v-slot:footer</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;<span data-darkreader-inline-color="">p</span>&gt;这是来自父组件的页脚内容&lt;/<span data-darkreader-inline-color="">p</span>&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/<span data-darkreader-inline-color="">template</span>&gt;<br>&nbsp;&nbsp;&lt;/<span data-darkreader-inline-color="">ChildComponent</span>&gt;<br>&lt;/<span data-darkreader-inline-color="">template</span>&gt;
```

在父组件中，我们使用 `v-slot` 指令（也可以简写为 `#`）来指定要填充哪个具名插槽，并在其中提供相应的内容。这样，当渲染子组件时，就会在对应的位置插入这些内容。

### vue3 使用具名插槽并且传递值

在 Vue 3 中，具名插槽不仅可以用于传递 HTML 结构，还可以配合作用域插槽（Scoped Slots）来传递数据。作用域插槽允许子组件向其插槽内传递数据，而父组件则可以通过插槽提供的回调函数来访问这些数据并在插槽内进行渲染。
